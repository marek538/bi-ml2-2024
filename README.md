# 1) Fashion MNIST: Dimensionality Reduction & Binary Classification

## 📌 Project Description
This project focuses on the binary classification of high-dimensional image data (28x28 grayscale images of clothing items like T-shirts and Trousers from the Fashion MNIST dataset). 

The primary goal is to address the "curse of dimensionality" by applying and comparing various dimensionality reduction techniques and traditional machine learning algorithms to find the most accurate and efficient predictive model.

# 2) Fashion MNIST: Multi-Class Neural Network Classification

## 📌 Project Description
A deep learning project implementing and comparing different neural network architectures to perform multi-class classification on the Fashion MNIST dataset. The goal was to optimize model performance through systematic architectural tuning and training strategy experimentation.

## 🚀 Solution & Algorithm
The project utilizes the **PyTorch** framework to explore various neural network configurations
